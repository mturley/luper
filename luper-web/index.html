Installation Instructions

  1. Download LuperAlpha.apk and copy it to your Android device's storage.
  2. Make sure your device is set to allow installation of applications from unknown sources (found in Settings > Security)
  3. Make sure you have a File Manager application on your device, such as ASTRO (available on Google Play)
  4. Using your File Manager, navigate to and open the LuperAndroid.apk file. You will be prompted to install the application.
